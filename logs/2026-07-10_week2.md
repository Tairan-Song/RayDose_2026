## Key Result Summary

The first functional baseline is complete, but performance is weak.

| Metric | Mean |
| --- | ---: |
| masked relative MAE | 13.12% |
| masked RMSE | 1.63e-05 |
| high-dose MAE, target >= 10% max dose | 2.46e-05 |
| high-dose MAE, target >= 50% max dose | 4.92e-05 |
| gamma pass rate, 3%/3mm, 10% cutoff | 7.18e-07 |
| gamma pass rate, 2%/2mm, 10% cutoff | 0.0 |
| prediction time | 0.112 s/sample |
| total time | 0.322 s/sample |

## Interpretation

The baseline is functional and fast, but it underfits. The near-zero gamma pass
rate shows that the predicted spatial dose distribution is not yet accurate.

The current model is useful as a technical baseline:

- the data pipeline works
- training runs on GPU
- checkpoints are created
- validation predictions are exported
- `Dose_Bx_CPy.mha` files are generated
- dose-specific metrics are produced

It is not yet a strong dose prediction model.
